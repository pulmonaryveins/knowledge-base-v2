Pre-Press Checklist

Non-negotiable requirements before sending any file to press.

✓300 DPI minimum — all images and raster elements
✓CMYK colour profile — convert from RGB before export
✓3mm bleed on all edges — artwork extends past trim
✓Fonts outlined or embedded — no missing typeface errors
✓PDF/X-1a export — industry-standard press-ready format
